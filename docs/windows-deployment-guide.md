# Archived: Windows Deployment Guide

Last updated: 2026-06-18

이 문서는 과거 Windows 중심 배포/운영 기준 문서의 보관 위치다. 현재 배포 기준으로 사용하지 않는다.

현재 기준:

- Windows는 Codex/Claude Code 실행 및 파일 편집 환경
- 실제 구현/검증/배포는 Linux/Ubuntu 기준
- runtime data는 실제 PostgreSQL / MongoDB / FTP / SAML 기준

현재 배포 절차는 `docs/ubuntu-deployment-guide.md`를 따른다.
